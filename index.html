📚 Project Information

  • Total Pages：6 core pages + 1 index page = 7 pages
  • Supported Roles：TA, MO, Admin (profile/basic access)
  • Tech Stack：HTML5 + CSS3 + Vanilla JavaScript + Java Servlet backend
  • Sprint 1 Features：Login/Logout, TA profile + resume upload, MO post positions, TA apply, MO review applicants

  • Interaction Design：hover animation、transition effect、loadStatus、mistakeTip、Successfeedback
